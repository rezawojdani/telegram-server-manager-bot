# 🚀 Telegram Server Manager Bot

![GitHub stars](https://img.shields.io/github/stars/rezawojdani/telegram-server-manager-bot?style=social)
![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Platform](https://img.shields.io/badge/platform-Ubuntu%20%7C%20Debian-orange)

A powerful Telegram bot to manage your Linux server remotely — no panel, no web UI, just Telegram.

## ✨ Features

- 📊 Live Dashboard — CPU, RAM, Disk, Network speed
- 💻 Terminal Mode — run any shell command via Telegram
- 🛡️ UFW Firewall — enable/disable/add/delete rules
- 🐳 Docker Manager — list, start, stop, restart, logs
- 👥 User Manager — add/delete/change password
- ⚡ Speedtest — animated real-time speed test
- 🔔 CPU Alert — background alert when CPU exceeds threshold
- 🚨 SSH Login Alert — instant Telegram message on every SSH login

## ⚡ One-Line Install

```bash
wget https://raw.githubusercontent.com/rezawojdani/telegram-server-manager-bot/main/install.sh && chmod +x install.sh && ./install.sh
```

## 📋 Requirements

- Ubuntu / Debian Linux
- A Telegram Bot Token (from [@BotFather](https://t.me/BotFather))
- Your Telegram User ID (from [@userinfobot](https://t.me/userinfobot))

## 🔄 Update / Remove

Re-run the installer and choose:
- Option `1` to reinstall / update
- Option `2` to completely remove
